Axisbits Test
===

Axisbits Test site theme.

* Use SCSS for styles
* Gulp + Webpack tasks for minimize JS / SCSS

Installation
---------------

### Requirements

`Axisbits_Test` requires the following dependencies:

- [Node.js](https://nodejs.org/)
### Setup

To start using all the tools that come with `Axisbits_Test`  you need to install the necessary Node.js dependencies :

```sh
$ npm install
```

### Available CLI commands

`Axisbits_Test` comes packed with CLI commands tailored for WordPress theme development :
- `npm run gulp` : checks all JavaScript / SCSS / IMAGES / FONTS and copy, minimize them
- `npm run build` : PRODUCTION ready command for minimize and optimize all ASSETS.
